pprint 模块

–> 模块描述：

pprint 是 Data pretty printer 的意思，该模块提供了将任意 Python 数据结构格式化（Pretty）打印的方法，可以提高输出数据的可读性。

1. PrettyPrinter Class

pprint 模块中定义有一个类：

1–> indent ：控制每层嵌套之间的空格数量。默认为1。

2–> width：控制打印显示的宽度，默认为 80 个字符。注意：当单个数据对象的长度超过 width 时，并不会分多行显示，而是会突破规定的宽度。

3–> depth：控制数据打印出来的层级，如果超过了规定的层级，会以 … 显示。注意，默认是不约束层级数量的。

3–> stream：设置输出流，模块中默认使用 sys.stdout。该对象唯一会使用到的函数是 write()。

4–> compact：3.4 版本之后加入的参数，默认为 False。如果值为 False，则超过 width 规定长度的序列会被分散打印到多行。否则（True），会尽量使序列填满 width 规定的宽度。通常需要和 width 参数配合使用。

PrettyPrinter 对象常用方法：

1.1 PrettyPrinter.pprint(object)
1.2 PrettyPrinter.pformat(object)

用法与 PrettyPrinter.pprint 相同。

不同的是，PrettyPrinter.pformat 函数格式化之后返回的类型是字符串。

1.3 Other Methods

1）PrettyPrinter.isreadable(object)

判断对象是不是可读（对象可以用 eval() 函数运行）。比如对象中有别的自定义对象时，或者包含了自身时，返回False。

2）PrettyPrinter.isrecursive(object)

判断对象是不是包含自身（是否递归）。

更多方法见：PrettyPrinter。

–> 总结:

pprint 是一个非常好用的模块，可以用来向文件、日志、控制台输出格式化后的数据，并且可以控制输出的格式。

经过 pprint 格式化后的数据更加易读、易理解，比起 Python 内建函数 print 更加 Pretty。
