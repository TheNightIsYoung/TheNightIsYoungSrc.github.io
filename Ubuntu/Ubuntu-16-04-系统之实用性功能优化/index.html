2. 快捷截图工具

一般用到的截图类型有三种：全屏、当前活动窗口、自定义区域。其中自定义区域截图是最灵活也是我们用的最多的方式。

Ubuntu 自带的截图工具其实已经能够满足最基本的截图功能，可以不必安装另外的截图软件。

Ubuntu 自带的截图工具为 screenshot，可以在 Dash home 中搜索找到（如下图），打开之后就可以进行三种基本截图功能。
